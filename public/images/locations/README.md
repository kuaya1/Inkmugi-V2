# Location Hero Images

Place optimized hero images in this directory for each location page.

## Naming Convention

- `{city-slug}-hero.webp` — e.g., `annandale-hero.webp`

## Required Specs

- **Dimensions:** 1600×900px
- **Format:** WebP
- **Max size:** 200KB
- **Content:** Ombre powder brows results, PMU artist working, natural eyebrow close-ups, or studio environment

## Temporary Fallback

Until local images are added, pages use Flickr-hosted portfolio images as hero images.
Replace Flickr URLs in each page's `heroImage` prop with local paths when ready:

```
heroImage="/images/locations/annandale-hero.webp"
```
